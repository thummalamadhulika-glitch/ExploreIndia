# Explore India - Travel Website

## Project Description
Explore India is a travel website created using HTML, CSS, and JavaScript. It allows users to explore popular tourist destinations, view travel packages, browse a gallery, and contact the travel agency.

## Features
- Home Page
- About Page
- Packages Page
- Gallery Page
- Contact Page
- Responsive Design
- JavaScript Form Validation

## Technologies Used
- HTML5
- CSS3
- JavaScript
